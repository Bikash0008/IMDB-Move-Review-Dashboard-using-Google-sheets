# IMDB-Move-Review-Dashboard-using-Google-sheets
The dashboard provides a clear overview of the movie industry using IMDB data. It highlights dominant actors, top performing movies, highly rated directors, and production trends. The analysis also shows that the USA dominates movie production and that movie releases have increased significantly in recent decades.
